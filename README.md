# To-Do List Application (Python CLI)

This is a simple console-based To-Do List Application created as part of Elevate Labs Python Developer Internship (Task 2).

## Features
- Add new tasks
- Remove tasks
- View all tasks
- Saves tasks in a text file (persistent storage)

## Technologies Used
- Python
- File Handling
- Lists
- String Manipulation

## How to Run
1. Clone or download the project.
2. Open terminal or VS Code.
3. Run the script:

   python todo.py

4. Choose options from the menu.

## Files
- todo.py → Main program
- tasks.txt → Stores tasks
